# 創建和刪除操作

## touch
- 創建檔或修改檔時間
- 如果檔案不存在，可以創建一個空白檔
- 如果檔案已經存在，可以修改檔的末次修改日期

---
## mkdir
- 創建一個新的目錄

| 選項	 | 含義 |
| :--------:| -------- | 
| -p    |可以遞迴創建目錄   | 
>新建目錄的名稱不能與目前的目錄中已有的目錄或檔案同名

---
## rm

使用 rm 命令要小心，因為檔刪除後不能恢復
| 選項	 | 含義 |
| :--------: | -------- | 
| -f    |強制刪除，忽略不存在的檔，無需提示|
|-r|遞迴地刪除目錄下的內容，刪除資料夾 時必須加此參數
