# a001: Hello! (*)
###### tags: `c語言題庫`

內容：
向人問好是最基本的禮貌，在台灣我們遇到人的時候會說「你好」，在日本遇到人的時候會說「こんにちは」，而在不知道對方國家的語言時，我們可以說「Hello!」。

輸入說明：
(無)
輸出說明：
請輸出 Hello!
範例輸入：help

(無)
範例輸出 ：

Hello!
提示 ：
本題範例程式：

#include  <iostream>

using namespace std;

int main()
{
    cout << "Hello!" << endl;
    return 0;
}
