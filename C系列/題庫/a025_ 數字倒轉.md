# a025: 數字倒轉
###### tags: `c語言題庫`
內容：
給你一個正整數，請你將所有數字倒轉之後輸出，例如給你12345，則輸出54321。
輸入說明：
輸入一個正整數 N。
輸出說明：
將 N 的所有數字倒轉之後輸出，開頭的 0 請不要輸出。
範例輸入：help

輸入1:
12345

輸入2:
1997
範例輸出 ：

輸出1:
54321

輸出2:
7991
提示 ：
可以用另一個變數 M 去記錄倒轉之後的數字是多少，觀察下面 N、M 兩個變數的變化：

N -> 12345 -> 1234 -> 123 ->  12  ->    1   ->     0
M ->    0     ->     5   ->  54 -> 543 -> 5432 -> 54321
